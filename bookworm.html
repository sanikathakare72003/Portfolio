Bookworm – Online Bookstore

Developed a web-based platform for purchasing and renting eBooks and audiobooks. The platform provides a seamless experience for readers to browse, purchase, and rent digital content.

Key Features

  • User authentication & secure login
  • Royalty calculation for content beneficiaries
  • My Shelf – time-bound access for rented content
  • Purchase & rental of eBooks and audiobooks
  • Microservices-driven backend architecture
  • Dual backend: Spring Boot 3 & .NET Core 8.0

Tech Stack

  • Frontend: React JS
  • Backend 1: Spring Boot 3, REST APIs, JPA/Hibernate, JWT Security, Microservices
  • Backend 2: .NET Core 8.0, Entity Framework Core
  • Database: MySQL 8
